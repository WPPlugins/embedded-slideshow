== Description ==

This plugin help you to insert a Picasa embedded slideshow to your post or page easily.
This plugin use the WordPress shortcode features, but you don't need to write even one
character, the plugin is designed user friendly and you insert a flash slideshow to your
post or page through just a few clicks.

The slideshow gadget is a flash provided by Google Picasa Web Album. The pictures or
photos in a slideshow should be put in one of the Picasa web album.

To insert a sexy slideshow to your post, there are three steps you have to do:

1. Create a new album in Picasa Web and upload the photos or pictures you want to slideshow.
2. Click the Picasa logo like button on top of the text editor in the Post New View of WP.
3. Choose an album by click the album cover in the list showed by this plugin.

== Installation ==

Install this plugin and get it work is simply easy. Just download the zip file, and
uncompressed it, then upload it to the `/wp-content/plugins/` directory. Activate
it. Done!
